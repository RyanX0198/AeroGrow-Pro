## 🧪 核心技术

- **PPFD 计算**: `PPFD = 600 × B% × (30/d)²`
- **距离反平方定律**: 可视化光强随距离变化
- **DLI 智能闭环**: 自动补光算法
- **AI 病虫害诊断**: 边缘AI识别叶片病害
